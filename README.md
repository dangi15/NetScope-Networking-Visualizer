A network visualiser in C++ created using SFML

HOW TO USE:
1. Create nodes using left click
2. To establish a connection between two nodes, right click the first node, then the second node. Then, enter weight of connection and press ENTER to finalise.
3. Succesfull connection is shown by both the respective nodes turning green and the connection between them is shown by a line and its respective weight.
4. To use BFS to find the path between two nodes with minimum nodes between them, press B, and right click the first node, then the second node. The final path is highlighted
5. To use Dijkstra's to find the shortest path between two nodes, press D, then right click the first node, then the second node. The final path is highlighted.
6. To reset the workspace and work on a new network, press C.
